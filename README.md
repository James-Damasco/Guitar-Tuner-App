# Guitar-Tuner-App
This **Guitar-Tuner-App** is an easy-to-use tool for guitarist who want to tune their guitar quickly and accurately. it uses the device's microphone and <a href="https://developer.mozilla.org/en-US/docs/Web/API/Web_Audio_API" style="color: black; text-decoration= none; cursor: pointer;">`WebAudio API`</a> to detect the sound of each string in real-time. </br>

The app displays the current note and frequency of the sound, as well as whether it is flat, sharp, or in tune. It also has a moving visual meter to make it easier to see how close the string is to the correct tuning.</br>

There is an auto-detect feature where it automatically detects which string is being played, and a manual mode if you want to select a specific string. Reference tones are also included so you can tune by ear.</br>

Furthermore, the design is responsive so it can be used on a phone or computer. The interface is simple and clean to avoid confusion, especially for beginners.
Overall, this app is a practical and easy-to-use guitar tuner that can be used by anyone, from beginners to intermediate guitarists.

# 🎸 Core (Basic) Features
### 1. 🎤 Microphone Input Detection
* Uses device mic to capture sound
* Detects frequency of the string in real-time

### 2. 🎵 Pitch Detetion (Frequency -> Note)
* Converts sound into:
* Note (E, A, D, G, B, E)
* Frequency (Hz)
* Shows if string is:
* Too low (flat)
* Too high (sharp)

### 3. 📊 Real-Time Visual Feedback
* Needle Meter / Bar Indicator
* Color Feedback (green = in tune)

### 4. 🔁 Auto String Detection
* Automatically detects which string is being played

### 5. 🎯 Manual Mode (Optional)
* User selects string manually
* Helpful for noisy environment
